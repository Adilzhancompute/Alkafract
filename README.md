# QSPR Модель для предсказания показателя преломления алканов

Этот проект содержит код для построения QSPR (Quantitative Structure-Property Relationship) модели,
использующей молекулярные дескрипторы RDKit и алгоритм Random Forest для предсказания показателя преломления алканов.

## Содержание:
- `QSPR_Refractive_Index.ipynb`: Основной Jupyter Notebook с кодом.
- `README.md`: Описание проекта.

## Как запустить:
1. Установите Python, Anaconda, Git.
2. Установите необходимые библиотеки: `pip install pandas numpy scikit-learn matplotlib seaborn rdkit`.
3. Запустите Jupyter Notebook и откройте `QSPR_Refractive_Index.ipynb`.
